# what's awada

awada是一个由LLM驱动的 **可在线自主学习的个人AI助理** 

只需将它拉入指定的微信群聊或者为它增加关注的公众号，它就可以自动分析并“学习”群聊内或者公众号发布的任何信息，这些信息可以广泛的在下游应用于各类RAG任务。

同时它也支持指定新闻类站点监控，进行自动搜集并学习。

<img alt="scope" src="asset/what&#39;sawada.png" width="960"/>

awada项目承诺开源，并欢迎被各其他RAG项目集成，为此我们设计了简洁清晰的接口，并支持“插件”架构。

awada项目面向个人用户单机使用永远免费，商用或复杂部署需求请邮件联系 **35252986@qq.com**
